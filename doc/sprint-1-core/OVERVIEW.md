# Sprint 1 — Ядро

**Мета:** Створити працездатний прототип гри — дошка відображається в терміналі, гравець може робити ходи, AI відповідає.

**Тривалість:** 1 тиждень

**Ключові результати:**
- Структура проєкту з pyproject.toml, залежності встановлюються через `uv sync`
- pytest налаштований, базові smoke-тести проходять
- GitHub Actions CI pipeline (ruff + mypy + pytest) на кожен push/PR
- Автоматичне AI-ревью PR через `ai-reviewbot` (Google Gemini)
- Генератор символів X/O у вигляді ASCII-арт блоків 17x7
- Rich-рендерер дошки з подвійними рамками та координатами
- Обробка введення у форматі a1..c3
- Game Runner — запуск повного ігрового циклу (Human vs AI)

**Таски:** 5 | **Пріоритет:** High
