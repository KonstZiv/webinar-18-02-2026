# Sprint 3 — Анімації

**Мета:** Реалізувати всі візуальні ефекти — від появи символу до кінематографічного падіння фігур з фізикою обертання.

**Тривалість:** 1.5-2 тижні

**Ключові результати:**
- Place Effect — анімація появи символу (typewriter)
- Highlight — мигання виграшної лінії зеленим
- Rotation sprites — набори кадрів обертання X/O (4+ фаз)
- 2D фізика — гравітація, колізії з рамками, бічні імпульси
- Falling Symbols — оркестрація каскадного падіння з обертанням
- Draw Effect — розчинення символів при нічиї
- Result Banner — ASCII-арт банер переможця

**Таски:** 7 | **Пріоритет:** High/Medium
