# Task 15 — Result Banner: Детальний опис

## Мета
Створити фінальний банер з результатом гри у вигляді великого ASCII-арт тексту, стилізованого відповідним кольором.

## Що робимо

1. Створити `src/tictactoe_cli/animations/celebration.py`
2. **Тести:** додати `tests/test_banner.py` — перевірка що банер містить mark переможця, не crash
3. Функція `show_result_banner(console, game_state)`:
   - Визначити результат (перемога X, перемога O, нічия)
   - Вивести великий ASCII-арт текст
   - Стилізувати кольором

### Підхід
- Банери зберігаються як константні рядки в модулі
- Альтернатива: використати бібліотеку `pyfiglet` для генерації (але вона додає залежність)
- Рекомендація: вручну створити 2-3 банери, зберегти як константи

### Стилізація
- Перемога: `[bold green]` + рамка з `Panel(border_style="green")`
- Нічия: `[bold yellow]` + рамка з `Panel(border_style="yellow")`
- Банер переможця включає символ гравця (X або O) під основним текстом

### Анімація появи (опціонально)
- Банер може з'являтися порядково (typewriter) для драматичного ефекту
- Або з fade-in: `dim` → `` → `bold`
- Або без анімації — достатньо статичного виводу

### Центрування
- Банер центрується горизонтально за шириною терміналу
- `rich.align.Align.center()` або `console.width // 2 - banner_width // 2`

## Очікувані результати

- При перемозі — великий зелений банер "ПЕРЕМОГА" з символом переможця
- При нічиї — великий жовтий банер "НІЧИЯ"
- Банер центрований і чітко читається
- Банер з'являється після завершення анімацій (falling/draw)
