### Запис GIF
1. Встановити `asciinema`: `brew install asciinema`
2. Записати сесію: `asciinema rec demo.cast`
3. Зіграти демо-партію (бажано з перемогою для falling symbols)
4. Конвертувати в GIF: `agg demo.cast demo.gif`
   - Або використати `terminalizer` / `vhs` від Charmbracelet
5. Помістити в `assets/demo.gif`

### Альтернатива GIF — SVG через asciinema
- `asciinema` підтримує вбудовуваний SVG-плеєр
- Менший розмір файлу, краща якість
- Але потребує JS для програвання

### Розмір GIF
- Ціль: < 5MB (GitHub рекомендація)
- Оптимізувати: менша роздільність, менше кольорів, обрізати зайві кадри
- Інструмент: `gifsicle` для оптимізації

## Очікувані результати

- README — інформативний, привабливий, з GIF на початку
- Інструкції з встановлення працюють на чистій машині
- CI badge відображається і оновлюється
- GIF демонструє: запуск → меню → гра → анімації → результат

## Як тестувати виконання

1. Відкрити README на GitHub — перевірити рендеринг markdown
2. GIF програється автоматично і показує геймплей
3. Скопіювати команди з Installation — вони працюють
4. Badge показує зелений статус

## Контрольні точки поза тестуванням

1. README не перевантажений — 200-300 рядків максимум
2. GIF < 5MB, програється без затримок
3. Markdown коректно рендериться на GitHub (таблиці, код-блоки, зображення)
4. Інструкції написані для аудиторії, що вперше бачить проєкт
